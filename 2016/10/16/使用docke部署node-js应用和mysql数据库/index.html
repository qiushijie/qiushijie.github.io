镜像安装

从docker hub下载镜像并安装，如果遇到权限不足情况加上sudo运行，使用latest会自动安装最新的版本，并且会将本地镜像自动更新到最新。

生成容器

mysql

生成一个名为mysql的mysql容器，设置密码为my-secret-pw，默认用户名为root，并设定编码为utf-8，更多设置查看镜像文档

node.js

生成一个名为node的node.js容器，并link mysql容器到当前容器，别名为db，将node容器内部3000端口映射到主机的3333端口。

使用link命令可以在两个容器将建立连接，主容器可以直接访问子容器中的网络端口。

mysql进行将会在子容器中自动设定其密码和host以及post，在程序中使用process.env获取相关的环境变量
