使用antlr编写一个json解析器

Veröffentlicht am 2016-10-06   |   in 编译原理   |  

文法定义

根据json的语法格式，写出一个json的antlr文法。

json

json由一个object({…})或者array([…])组成，文法是

object

object以一个’{‘符号开始，以一个’}’符号结束，由一个或者多个键值(key : value, …)对组成，也可以为空，文法是

array

array以’[‘符号开始，’]’符号结束，由一个或者多个值(value, …)组成，也可以为空，所以文法定义是

pair

一个键值对由一个字符串作为键，一个值组成，文法定义是

value

一个值可以是字符串，也可以是数字，也可以是一个bool，也可以是null。值还可以是一个object或者array，表示嵌套的json格式。文法定义是

STRING

字符串由’”‘开始，’”‘结束，’\”‘是反义符号，在字符串内表示‘“’

小结

编译原理在编程中有很广泛的应用，在java中，像hibernate等的orm框架使用antlr来解析sql语句，在beelt模板引擎中使用antlr来解析模板文件。在外部dsl、代码生成、静态分析等领域也有很广泛的应用。antlr的文法比较简单，入门门槛也不高，编译原理没有想象中这么困难。
