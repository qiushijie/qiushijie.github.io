docker remote api在node.js中使用

发表于 2016-10-26   |   分类于 docker   |  

docker提供了remote api供远程调用，可用于镜像部署，容器操作等大部分docker命令行提供的操作。docker remote api使用socket进行通信，官方文档(v1.24)中有api参数的详尽说明。第三方开发者提供了各种语言的api实现库，可以直接使用这些实现库来进行操作。node.js的api实现库是dockerode，docker可视化管理工具kitematic也是使用了dockerode来进行docker的操作。
